# Compliance Notes

This folder tracks compliance obligations for CommutePool v1 operating in Telangana.

## Key Requirements

- Grievance officer details must be visible on app and website
- Support contact must be accessible from the app
- Incident/SOS handling with escalation must be operational before beta
- Audit trail for admin-sensitive actions (suspension, verification, incident resolution)
- Contribution cap policy must be documented and accessible

## References

- Motor Vehicles Aggregator Guidelines 2025 (MoRTH)
