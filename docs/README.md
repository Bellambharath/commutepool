# CommutePool Documentation

This folder contains all frozen contracts, architecture decisions, runbooks, and setup guides.

```
docs/
├── contracts/      # All frozen implementation contracts (source of truth)
├── decisions/      # Architecture decision records (ADRs)
├── setup/          # Local dev, environment, and deployment setup
├── runbooks/       # Beta ops, incident, support, corridor runbooks
└── compliance/     # Grievance, legal, and regulatory notes
```
