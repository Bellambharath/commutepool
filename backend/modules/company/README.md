# Company Module

Owns company tenant, work-email verification, company membership, and corridor restrictions.

## Commands
- InitiateOfficeEmailVerification
- ConfirmOfficeEmailVerification
- AssignUserToCompany
- SetCompanyEnterpriseMode

## Queries
- GetCompanyByDomain
- GetUserCompanyMembership
- GetCompanyDashboardSummary

## Events
- OfficeEmailVerificationStarted
- OfficeEmailVerified
- CompanyMembershipApproved
- CompanyTenantUpdated
