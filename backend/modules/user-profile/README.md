# User Profile Module

Owns user account, role mode, emergency contact, preferences, and block list.

## Commands
- CreateProfile
- UpdateProfile
- UpdateEmergencyContact
- BlockUser
- UnblockUser

## Queries
- GetMyProfile
- GetUserSummary
- GetBlockedUsers

## Events
- ProfileCreated
- ProfileUpdated
- EmergencyContactUpdated
- UserBlocked
