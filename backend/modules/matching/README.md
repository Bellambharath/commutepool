# Matching Module

Owns candidate generation, scoring, ranking, expiry, and recurring pair activation.

## Commands
- GenerateMatchesForRequest
- GenerateMatchesForOffer
- AcceptMatch
- RejectMatch
- ExpireMatch
- CreateRecurringPair

## Queries
- GetMatchesForUser
- GetMatchDetail
- GetRecurringPairs

## Events
- MatchGenerated
- MatchAccepted
- MatchRejected
- MatchExpired
- RecurringPairCreated
