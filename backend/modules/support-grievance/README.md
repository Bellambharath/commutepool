# Support & Grievance Module

Owns support tickets, complaint workflows, and grievance handling.

Grievance officer contact details must be visible on app and website per compliance requirements.

## Commands
- CreateSupportTicket
- AssignSupportTicket
- ReplyToSupportTicket
- ResolveSupportTicket

## Queries
- GetMySupportTickets
- GetSupportQueue

## Events
- SupportTicketOpened
- SupportTicketAssigned
- SupportTicketResolved
